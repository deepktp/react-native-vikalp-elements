## Props

<div class='table-responsive'>

| Name              | Type                                                                   | Default          | Description                                                               |
| ----------------- | ---------------------------------------------------------------------- | ---------------- | ------------------------------------------------------------------------- |
| `checked`         | boolean                                                                |                  | Flag for checking the icon.                                               |
| `checkedColor`    | string                                                                 |                  | Default checked color.                                                    |
| `checkedIcon`     | `string` \| `ReactElement<{}, string` \| `JSXElementConstructor<any>>` | `check-square-o` | Default checked icon ([Font Awesome Icon](http://fontawesome.io/icons/)). |
| `iconType`        | IconType                                                               |                  | Type of icon set. [Supported sets here](icon#available-icon-sets).        |
| `onIconPress`     | Function                                                               |                  | onPress function for checkbox.                                            |
| `onLongIconPress` | Function                                                               |                  | onLongPress function for checkbox.                                        |
| `size`            | number                                                                 | `24`             | Size of the checkbox.                                                     |
| `uncheckedColor`  | string                                                                 | `#bfbfbf`        | Default unchecked color.                                                  |
| `uncheckedIcon`   | `string` \| `ReactElement<{}, string` \| `JSXElementConstructor<any>>` | `square-o`       | Default checked icon ([Font Awesome Icon](http://fontawesome.io/icons/))  |

</div>
